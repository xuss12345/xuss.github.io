指针的意义

指针是一种变量，是专门用来存放其他类型变量的地址。指针变量在64位系统中占八个字节，在32位系统中占四个字节。

一个内存单元的位置叫做内存单元的地址，通常用取地址符 & 表示，scanf中&a表示将键盘输入的内容存放到变量a的地址中:

scanf("%d",&a);

这在现实中可以用房屋的地址来理解，我们可以说这一栋房子是小李家，也可以说一栋房子是xx路xxx号（指针表示）。

指针变量的声明

定义指针类型的方法:将声明符写成*d的形式，其中d是变量名。在一条语句内声明多个指针对象必须每个变量名前都有*。

例子

  int *ip1,*ip2;//ip1和ip2都是用来存int类型变量的地址的指针
  double dp,*dp2;//dp2是指向double型对象的指针,dp是double型对象(不一样!!!)

正确示例

double dval;
double *pd=&dval;
double *pd2=pd;

错误示例

double dval;
double *pd = &dval;
int *pi = pd; // 错误：指针pi的类型和pd的类型不匹配
pi = &dval; // 错误：试图把double型对象的地址赋给int型指针

空指针

空指针不指向任何对象，在试图使用一个指针前可以先检查指针是否为空。

得到空指针最直接的办法就是用字面值nullptr来初始化指针。

直接把int变量赋值给指针是错误的操作，即使int变量的值恰好等于0也不行。

// 空指针的创建方法
int *p1 = nullptr;
int *p2 = 0;
int *p3 = NULL; // 等价于 int *p3 = 0;
// 不能直接将int变量赋值给指针哟!!!

指向指针的指针

指针是一种特殊的变量，有自己的地址，允许把指针的地址再存放到另一个指针中。另一个指针就是指针的指针。

通过*的个数可以区分指针的级别，即为**表示指向指针的指针，***表示指向指针的指针的指针。

#include <iostream>using namespace std;
int main()
{
	// 指向指针的指针
	int ival = 1024;
	int *p1 = &ival;
	int **pp1 = &p1;
	cout << "ival = " << ival << endl;
	cout  << "*p = " << *p1 << endl;
	cout << "**pp = " << **pp1 << endl;
	return 0 ;
}

指针的基本操作

指针的基本操作

对于指针来说，解引用和取地址是最重要的两个操作符。

#include <bits/stdc++.h>using namespace std;
int main()
{
	double dval=3.14;
	double *pd=&dval;
	double *pd2=pd;
	cout<<"dval的地址:"<<&dval<<endl<<"pd:"<<pd<<endl<<"*pd:"<<*pd<<endl<<"pd2:"<<pd2<<endl<<"*pd2:"<<*pd2;
	return 0;
}

#include <iostream>using namespace std;
int main()
{
	int num=4;
	int *intPtr = &num;
	cout<<"指针所指的数字是:"<<*intPtr<<endl;
	cout<<"num的值是:"<<num<<endl;
	*intPtr=3;
	cout<<"修改后,指针所指的数字是:"<<*intPtr<<endl;
	cout<< "num的值是:"<<num;
	return 0 ;
}

指针和数组

C++语言中，指针和数组关系密切。
数组还有一个特性：数组名字就是数组的首地址，也就是a[0]的地址。
#include<iostream>using namespace std;
//指针与整型的算数操作
int main()
{
	int arr[5] = { 0, 1, 2, 3, 4 };
	int *ptr = arr;
	cout<<"ptr:"<<ptr<<endl;
	cout<<"*ptr:"<<*ptr<<endl;
	cout<<"ptr + 4:"<<ptr+4<<endl;
	cout << "*ptr + 4:"<<*ptr+4;
	return 0;
}

// 利用指针遍历数组
