题目描述

由于先序、中序和后序序列中的任一个都不能唯一确定一棵二叉树，所以对二叉树做如下处理，将二叉树的空结点用·补齐，如图所示。我们把这样处理后的二叉树称为原二叉树的扩展二叉树，扩展二叉树的先序和后序序列能唯一确定其二叉树。

现给出扩展二叉树的先序序列，要求输出其中序和后序序列。

输入

一行字符串为扩展二叉树的先序序列，第个结点用一个字符表示。

输出

两行，第一行为原二叉树的中序序列，第二行为原二叉树的后序序列。

样例输入

ABD..EF..G..C..

样例输出

DBFEGAC
DFGEBCA

提示

【注意】

虽然题中二叉树结点不多（只有哪么多单一字符），但并没有说明二叉树的深度，因此不能用一维数组顺序存储建树，用静态二叉链表存储。

字符串长度小于等于1000

思路

递归建树，一个变量x记录当前节点序号，递归遍历，至于树的遍历详见

树(入门)-CSDN博客

具体细节详见代码

代码

#include <bits/stdc++.h>using namespace std;
int thekey=0;
struct tree
{
	int l,r;
	char data;
}tree[1100];
int build()
{
	char data;
	cin>>data;
	if(data=='.')
	{
		return 0;
	}
	else
	{
		int x=++thekey;//这步很重要，用thekey的话会随着递归而变化，就不是之前的值了
		tree[x].data=data;
		tree[x].l=build();
		tree[x].r=build();
		return x;
	}
}
void bl2(int step)
{
	if(step!=0)
	{
		bl2(tree[step].l);
		cout<<tree[step].data;
		bl2(tree[step].r);
	}
}
void bl3(int step)
{
	if(step!=0)
	{
		bl3(tree[step].l);
		bl3(tree[step].r);
		cout<<tree[step].data;
	}
}
int main()
{
	build();
	bl2(1);
	cout<<endl;
	bl3(1);
	return 0;
}
